## Usage

### Requirements

- [NodeJS](https://nodejs.org/en/)

### Run 

```bash
npm i 
```

```bash
npm start
```

### Example of code to analyze

```java
public void sum(int num1, int num2) {
	int sum;
	sum = num1 + num2;
	System.out.println(sum);
}
```
